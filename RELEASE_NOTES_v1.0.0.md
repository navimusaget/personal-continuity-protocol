# Release v1.0.0

Initial public release of the Personal Continuity Protocol Public Seed Package.

This release is aligned with the frozen Core Paper v1.0 archived on Zenodo:

https://doi.org/10.5281/zenodo.20759434

## Included

- Seed specification
- Cold Continuity Declaration
- Clause 32 template
- Continuity Notice template
- Emergency Maintenance Mandate template
- Continuity Fingerprint Sheet template
- Material Anchor Instruction
- Mnemonic Anchor Instruction
- Bare-Body Recovery Instruction
- Relay Agent Guide
- Event Handle Format
- Adversarial Anchor Map
- Anchor Class Independence Checklist
- Failure Modes and Boundaries note
- Medical Bridge / Hot Envelope note
- Blind Legacy Escalation note
- Notarial Deposit Cover Sheet
- `/.well-known/person-continuity` schema and example
- Witness Commons Event Record schema and example
- Minimal evidence-bundle generator

## Release model

The Core Paper is frozen. This Seed Package may evolve through future releases, annexes, implementation notes and forks.
